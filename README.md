## Getting Started

### Prerequisites

You need Android Studio to open this project _[Get Android Studio](https://developer.android.com/studio)_


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ericaskari/health-application.git
   ```
2. Open the project in Android Studio
